# Final Function Point Calculation - LouietexERP System

## 5.2.5 Final Calculation

### 1. Value Adjustment Factor (VAF)
The Value Adjustment Factor is calculated based on the Total Degree of Influence (TDI) from the 14 General System Characteristics.

*   **VAF = 0.65 + (0.01 × TDI)**
*   VAF = 0.65 + (0.01 × 54)
*   VAF = 0.65 + 0.54
*   **VAF = 1.19**

### 2. Total Unadjusted Function Point (UFP)
The total UFP is the sum of the transactional function points and the data function points.

*   **Total UFP = UFP (TF) + UFP (DF)**
*   Total UFP = 48 + 63
*   **Total UFP = 111**

### 3. Adjusted Function Point (AFP)
The final Adjusted Function Point represents the normalized size of the software, accounting for system complexity and technical requirements.

*   **AFP (Adjusted Function Point) = UFP (Total) × VAF**
*   AFP = 111 × 1.19
*   **AFP = 132.09**

The LouietexERP system is quantified at **132.09 Adjusted Function Points**. This metric provides a professional basis for estimating project effort, scheduling, and development costs for your final documentation.
