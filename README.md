LouietexERP
===========

Short description
-----------------
LouietexERP is an internal ERP web application scaffold built with ASP.NET Core (Razor Pages) targeting .NET 10. It provides basic order management UI and core project structure to continue development.

Status (so far)
---------------
- Initial project scaffold created targeting .NET 10.
- Basic UI and views present (e.g., Views/Orders/Index.cshtml).
- Most core features are in-progress; database and business logic need configuration.

Prerequisites
-------------
- .NET 10 SDK
- Visual Studio 2026 or VS Code
- Configure your database connection in appsettings.json or user secrets

Run locally
-----------
- Restore and build: dotnet build
- Run: dotnet run (or start from Visual Studio)

Next steps
----------

- Implement business logic and services
- Add authentication/authorization and tests
